# 🧠 MindSpace AI

MindSpace AI is an AI-powered emotional wellness companion built using React, TailwindCSS, Framer Motion, and Google Gemma/Gemini AI.

The platform provides supportive and empathetic conversations for users experiencing stress, anxiety, sadness, or emotional overwhelm through real-time AI interaction.

---

# ✨ Features

* 💜 AI-powered supportive mental wellness assistant
* 🎤 Voice input support
* 😊 Mood tracking system
* 🌈 Animated breathing orb for calming exercises
* 📊 Wellness score visualization
* 🧘 Stress-relief suggestion cards
* 📱 Fully responsive modern UI
* ⚡ Real-time Gemini/Gemma AI responses
* ☎️ Emergency mental health helplines

---

# 🚀 Tech Stack

* React + Vite
* TailwindCSS
* Framer Motion
* Google Gemini / Gemma AI
* Vercel Deployment

---

# 🧠 How Gemma AI Impacts This Project

MindSpace AI uses Google's Gemma/Gemini AI models to create emotionally aware and supportive conversations in real time.

Gemma AI enables the application to:

* Understand emotional tone from user input
* Generate empathetic and motivational responses
* Provide calming suggestions and wellness guidance
* Simulate supportive human-like interaction
* Deliver fast AI responses with lightweight infrastructure

The project demonstrates how AI can be used responsibly in mental wellness applications to provide emotional support accessibility at scale.

# 📸 Key Functionalities

## AI Emotional Support

Users can openly express feelings and receive short, safe, motivational AI responses.

## Breathing & Relaxation

Animated breathing orb encourages calm breathing exercises during stress.

## Mood Tracking

Interactive emoji-based mood selection improves emotional awareness.

## Voice Interaction

Users can interact using speech-to-text for accessibility and comfort.

## Emergency Resources

Integrated mental health helplines provide immediate support resources.

# 📦 Installation

```bash
git clone https://github.com/sowmyareddy278/mindspace-ai.git
cd mindspace-ai
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env.local` file:

```env
VITE_GEMINI_API_KEY=YOUR_API_KEY
```
